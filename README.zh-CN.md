# Hash🔎

## 项目简介

`Hash` 是一个基于 VuePress 和自定义主题的文档管理工具，旨在为开发人员提供一个简单、高效的文档管理平台。它支持快速构建和部署，并且具有本地预览功能，方便开发和生产环境的切换。

## 安装

使用 `pnpm` 安装依赖：

🛠️ **安装依赖**：

```bash
pnpm i
```

## 使用

### 启动开发服务

启动开发服务器以进行实时预览和调试：

🚀 **启动开发服务**：

```bash
pnpm docs:dev
```

### 构建生产包

构建生产环境下的文档包，准备部署：

📦 **构建生产包**：

```bash
pnpm docs:build
```

### 本地预览生产服务

构建后，可以使用以下命令在本地预览生产环境：

👁️ **本地预览生产环境**：

```bash
pnpm docs:project
```

### 更新 VuePress 和主题

更新 VuePress 和自定义主题到最新版本：

🔄 **更新 VuePress 和主题**：

```bash
pnpm dlx vp-update
```
